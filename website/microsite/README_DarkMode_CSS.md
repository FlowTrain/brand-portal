How to enable:
1) Place this file at `website/microsite/assets/css/style.css` in your repo.
2) Minima automatically includes `/assets/css/style.css` if present.
3) Commit & push; GitHub Pages will rebuild with dark styling.

Optional:
- If your theme does not load custom CSS, add the following to each page:
  `<link rel="stylesheet" href="assets/css/style.css">`
